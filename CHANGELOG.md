# 0.1.0

- `Mo2FACodeStyle.hintCharacter` / `hintStyle` — placeholder character shown
  in every empty cell.
- `Mo2FACodeStyle.separatorBuilder` — insert any widget between cells to group
  the code (e.g. `123 - 456`).
- `readOnly` — show the code without allowing edits.
- `hapticFeedback` — optional light haptic impact on every accepted character.
